# 中美关系新闻语料查询问题

可以用这些问题测试新闻语料检索效果。

## 基础检索

```text
特朗普访华实际达成了哪些成果？
```

```text
特朗普访华前中美双方最关注哪些议题？
```

```text
台湾军购案和特朗普访华之间有什么关系？
```

## 报告章节式问题

```text
请概述特朗普此次访华的背景：距离上一次访华相隔多久？访华前各方对议题有哪些预期和分析？
```

```text
请按时间顺序列出特朗普此次访华的关键场景，包括接机规格、参观中南海、国宴、与习近平会谈等。
```

```text
请总结此次会谈中双方关注的核心议题，特别是贸易、科技、台湾、伊朗、半导体等。
```

```text
此次访华实际达成了哪些共识或协议？哪些重要议题未能取得突破？
```

```text
不同媒体对此次访华的整体评价有何差异？“美国盯著 3B、中国盯著 3T”是什么意思？
```

```text
综合来看，此次访华对未来一到两年的中美关系会有什么影响？
```
